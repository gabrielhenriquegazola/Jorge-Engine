## What is JORGE?

JORGE implements the Deterministic Structural Observable (DSO),
a geometric operator designed to detect coherent structure in
3D spatial data without supervised learning.

Instead of semantic classification, JORGE measures:

- local density
- anisotropy
- directional coherence

and induces a global structural backbone from the regions of
highest structural relevance.

The system was validated on:

- LiDAR
- industrial E57 scans
- medical volumetric imaging
- large-scale spatial datasets

## O que é o JORGE?

O JORGE implementa o Observável Estrutural Determinístico (OED / DSO),
um operador geométrico projetado para detetar estruturas coerentes em
dados espaciais 3D sem necessidade de aprendizagem supervisionada.

Em vez de classificação semântica, o JORGE mede:

- densidade local
- anisotropia
- coerência direcional

e induz um backbone estrutural global a partir das regiões de
maior relevância estrutural.

O sistema foi validado em:

- LiDAR
- escaneamentos industriais E57
- imagens médicas volumétricas
- grandes conjuntos de dados espaciais
